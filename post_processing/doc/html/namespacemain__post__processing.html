Parameters
[in]Inputsare defined in the files in the "inputs" directory
  • post_processing.csv that contains
    – Data type: Type of data to plot, may be 'E' (E-field), 'F' (propag factor), or 'S' (Poynting vector)
    – Final field: Plot the last calculation plane. 'Y' or 'N'
    – Total field: Plot the total field from the wavelet parameters. 'Y' or 'N'
    – Dynamic: Difference between the min and max value plotted (in dB)
[out]Figuresare saved in the "outputs" directory

nota: the other parameters (frequency, wavelet, and so on) are read from the source and propagation modules
