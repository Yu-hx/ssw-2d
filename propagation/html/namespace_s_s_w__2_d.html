Detailed Description

The core of the 2D SSW: propagates a 1D initial field with SSW in 2D.

Date
06/02/19, modif 28/02/2022
Version
In progress

This function is the core of the 2D SSW code. It propagates a 1D initial reduced field with SSW in 2D, in a Cartesian environment. The output field is given in the wavelet domain for a reduced memory size.
The library of propagators is computed. Then the field is propagated in the wavelet domain using this library.
The refractivity is taken into account with a phase screen. Apodisation is also applied at each step.

Parameters
[in]u_0Initial field. Complex array of size (N_z)
[in]configClass that contains the configuration. See class list for details
[in]config_sourceClass that contains the source configuration. See class list for details
[in]n_refractionVector containing the modified refractive index n. Real array of size (N_z)
[in]z_reliefVector containing the relief indices at each distance step. Real array of size (N_z)
[out]U_totWavelet coefficients of the 3D field. Complex array of size (N_x, N_z)
Warning
Put lengths = multiple of 2^l
